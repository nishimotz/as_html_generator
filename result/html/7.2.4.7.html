7.2.4.7 視覚的に認識可能なフォーカスに関する達成基準 (等級AA)

実装方法およびテスト項目 対象 達成可否
C15-1: CSSを用いて、ユーザーインタフェースコンポーネントがフォーカスを受けとったときの表示を変更する a要素 達成可能
C15-2a: CSSを用いて、ユーザーインタフェースコンポーネントがフォーカスを受けとったときの表示を変更する input type="text" 達成可能
C15-2b: CSSを用いて、ユーザーインタフェースコンポーネントがフォーカスを受けとったときの表示を変更する textarea要素 達成可能
C15-2c: CSSを用いて、ユーザーインタフェースコンポーネントがフォーカスを受けとったときの表示を変更する input type="radio" 要注意
C15-2d: CSSを用いて、ユーザーインタフェースコンポーネントがフォーカスを受けとったときの表示を変更する input type="checkbox" 要注意
C15-2e: CSSを用いて、ユーザーインタフェースコンポーネントがフォーカスを受けとったときの表示を変更する select要素 達成可能
C15-2f: CSSを用いて、ユーザーインタフェースコンポーネントがフォーカスを受けとったときの表示を変更する input type="submit/reset/button" 要注意
C15-2g: CSSを用いて、ユーザーインタフェースコンポーネントがフォーカスを受けとったときの表示を変更する button要素 要注意
SCR26: 動的なコンテンツをDOMのそのトリガーとなる要素の直後に挿入する DOM操作でリンクの後に挿入される要素 達成可能
